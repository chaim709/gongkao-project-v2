# 公考管理系统 V2 - 项目总结

## 项目概述

公考管理系统 V2 是一个现代化的培训机构管理系统，专为公务员考试培训机构设计。系统采用前后端分离架构，提供学员管理、督学管理、课程管理、作业管理、智能选岗等核心功能。

**开发周期**：5 周（2026年2月-3月）
**项目状态**：✅ 已完成，可部署生产环境

---

## 技术栈

### 后端
- **框架**：FastAPI 0.109.0
- **数据库**：PostgreSQL + SQLAlchemy 2.0
- **认证**：JWT (python-jose)
- **迁移**：Alembic
- **日志**：structlog

### 前端
- **框架**：React 18 + TypeScript
- **UI 库**：Ant Design
- **样式**：Tailwind CSS
- **状态管理**：Zustand + TanStack Query
- **构建工具**：Vite

### 部署
- **容器化**：Docker + Docker Compose
- **Web 服务器**：Nginx
- **进程管理**：Uvicorn

## 核心功能清单

### ✅ 已完成功能

#### 1. 认证与权限
- JWT 认证系统
- 角色权限控制（admin/teacher）
- 登录/登出功能

#### 2. 学员管理
- 学员 CRUD（创建、查询、更新、软删除）
- 学员列表（分页、搜索、筛选）
- 学员信息字段：
  - 基础信息：姓名、手机、微信、性别、学历、专业
  - 扩展信息：家长电话、班级名称
  - 报考信息：报考类型、报名日期、状态
- 薄弱项标签系统（红黄绿三色标签）

#### 3. 薄弱项标签系统
- 30 个预设知识模块（行测/申论）
- 红黄绿三色标签（薄弱/一般/掌握）
- 点击切换掌握程度
- 正确率和练习次数记录
- 学员列表展开显示

#### 4. 督学管理
- 督学日志 CRUD
- 日志字段：
  - 日期、联系方式、联系时长
  - 心情状态、学习状态
  - 督学内容、下次跟进日期
- 常用短语快速插入
- 日期范围筛选

#### 5. 课程管理
- 课程 CRUD
- 课程信息：名称、描述、教师、开课时间

#### 6. 作业管理
- 作业 CRUD
- 作业字段：
  - 标题、描述、课程
  - 题量、知识模块
  - 截止时间
- 学员提交作业
- 教师批改作业（评分、反馈）
- 提交统计

#### 7. 打卡管理
- 打卡记录 CRUD
- 打卡统计

#### 8. 数据分析
- Dashboard 数据看板
- 学员统计图表
- 督学统计图表
- 作业完成率统计

#### 9. 智能选岗
- 岗位列表（分页、搜索）
- 岗位筛选（学历、专业、地区）
- 岗位详情查看

#### 10. 审计日志
- 操作日志记录
- 日志查询（分页、筛选）

## 数据库设计

### 核心表
- `users` - 用户表
- `students` - 学员表
- `module_categories` - 知识模块分类表
- `weakness_tags` - 薄弱项标签表
- `supervision_logs` - 督学日志表
- `courses` - 课程表
- `homework` - 作业表
- `homework_submissions` - 作业提交表
- `checkins` - 打卡表
- `positions` - 岗位表
- `audit_logs` - 审计日志表

### 数据库特性
- 所有表支持软删除（deleted_at, deleted_by）
- 关键操作记录审计日志
- 索引优化查询性能
- 外键约束保证数据一致性

## 部署指南

### 开发环境

**后端启动**：
```bash
cd backend
uvicorn app.main:app --reload
```

**前端启动**：
```bash
cd frontend
npm run dev
```

### 生产环境（Docker）

**一键部署**：
```bash
./deploy.sh start
```

**其他命令**：
- `./deploy.sh stop` - 停止服务
- `./deploy.sh restart` - 重启服务
- `./deploy.sh logs` - 查看日志
- `./deploy.sh backup` - 备份数据库
- `./deploy.sh update` - 更新代码并重启

**访问地址**：
- 前端：http://your-domain
- 后端 API：http://your-domain/api/v1
- API 文档：http://your-domain/api/v1/docs
