## 字段映射规则

### 1. 常见字段别名识别
请识别并映射以下常见的字段别名：

**年份相关**：年度、考试年份、报考年份
**考试类型**：考试名称、招考类型、考试类别
**地市**：地区、所在地市、市
**单位名称**：招录单位、用人单位、招考单位、部门
**职位名称**：岗位名称、职位、岗位
**招录人数**：计划招录、拟招录人数、招考人数、名额
**学历要求**：学历、最低学历、学历条件
**专业要求**：专业、专业类别、所学专业
**工作地点**：工作地、办公地点、地点
**报名人数**：报考人数、已报名人数
**竞争比**：竞争比例、报录比

### 2. 学历标准化（重要！）
无论原始数据如何表述，必须统一为以下三个标准值之一：

- **大专及以上**：包含"大专"、"专科"、"高职"、"大学专科"等
- **本科及以上**：包含"本科"、"学士"、"大学本科"、"本科学历"等
- **研究生及以上**：包含"研究生"、"硕士"、"博士"、"硕士研究生"、"博士研究生"等

**示例**：
- "本科" → "本科及以上"
- "大专或本科" → "大专及以上"
- "硕士研究生" → "研究生及以上"
- "本科及以上学历" → "本科及以上"

### 3. 数值字段处理
- **招录人数**：提取数字，如"1人" → 1，"若干" → 1
- **竞争比**：保留小数点后1位，如"75.5"
- **分数**：保留小数点后1位

### 4. 缺失字段处理
- 如果原始数据中没有某个字段，该列留空（不要填"无"或"-"）
- 必填字段缺失时，尝试从其他字段推断或使用合理默认值：
  - 年份：如果缺失，使用当前年份或从文件名推断
  - 考试类型：从文件名或上下文推断
  - 招录人数：默认为1

## 数据清洗规则

### 1. 文本清理
- 去除前后空格
- 统一全角/半角字符
- 去除多余的换行符和特殊字符
- 保留原始信息，不要过度简化

### 2. 地名处理
- 保持原始地名格式，不要随意简化
- 如"江苏省南京市" 保持不变，不要改成"南京"

### 3. 单位名称
- 保留完整的单位名称
- 不要缩写或简化

### 4. 专业要求
- 保留完整的专业描述
- 多个专业用顿号"、"分隔
- 如"财会类、经济类"

## 输出要求

### 1. 文件格式
- 输出为Excel文件（.xlsx格式）
- 第一行为表头（使用标准列名）
- 从第二行开始为数据
- 不要有合并单元格
- 不要有空行

### 2. 数据完整性
- 确保所有原始数据都被映射到标准格式中
- 如果原始数据有额外字段无法映射，在"其他要求"列中补充说明
- 不要丢失任何有价值的信息

### 3. 数据验证
整理完成后，请检查：
- [ ] 所有必填字段都有值
- [ ] 学历要求已标准化为三个标准值之一
- [ ] 数值字段格式正确（整数/小数）
- [ ] 没有明显的数据错误或遗漏
- [ ] 表头顺序与标准格式一致

## 特殊情况处理

### 1. 多个工作表
如果Excel文件包含多个工作表，请合并到一个工作表中，确保数据格式统一。

### 2. 表头在非第一行
如果表头不在第一行，请识别并提取正确的表头和数据。

### 3. 数据分散在多列
如果一个字段的信息分散在多列中，请合并到对应的标准列中。

### 4. 额外信息
如果原始数据包含标准格式中没有的字段，请评估：
- 如果是重要信息，添加到"其他要求"列
- 如果是统计信息，添加到对应的统计字段（如报名人数、竞争比等）

## 质量检查清单

整理完成后，请确认：
1. ✅ 表头完全符合标准格式（27列，顺序正确）
2. ✅ 所有必填字段都有数据
3. ✅ 学历要求已标准化（只有三个值）
4. ✅ 数值字段格式正确
5. ✅ 没有遗漏原始数据中的重要信息
6. ✅ 文本清理完成（无多余空格、特殊字符）
7. ✅ 数据行数与原始数据一致

---

**重要提示**：整理数据时，宁可保留更多原始信息，也不要随意删减或简化。如果不确定某个字段应该映射到哪一列，请在"其他要求"列中保留该信息。
