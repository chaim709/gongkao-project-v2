### 2. API 集成测试

#### 2.1 认证 API
```bash
# 登录
POST /api/v1/auth/login
# 获取当前用户
GET /api/v1/auth/me
```

#### 2.2 学员 API
```bash
# 列表
GET /api/v1/students?page=1&page_size=20
# 创建
POST /api/v1/students
# 更新
PUT /api/v1/students/{id}
# 删除
DELETE /api/v1/students/{id}
```

#### 2.3 薄弱项 API
```bash
# 获取知识模块
GET /api/v1/modules
# 学员薄弱项列表
GET /api/v1/students/{id}/weaknesses
# 创建薄弱项
POST /api/v1/students/{id}/weaknesses
# 更新薄弱项
PUT /api/v1/weaknesses/{id}
# 删除薄弱项
DELETE /api/v1/weaknesses/{id}
```

#### 2.4 督学 API
```bash
# 列表
GET /api/v1/supervision-logs
# 创建
POST /api/v1/supervision-logs
```

#### 2.5 作业 API
```bash
# 列表
GET /api/v1/homework
# 创建（含 question_count, module）
POST /api/v1/homework
# 提交作业
POST /api/v1/homework/{id}/submissions
# 批改作业
PUT /api/v1/homework/submissions/{id}/review
```
