# 测试执行指南

## 前置条件

1. 后端服务已启动
2. 数据库已初始化（包含种子数据）
3. Python 环境已安装 requests 库

## 测试步骤

### 1. 启动后端服务

```bash
cd backend
uvicorn app.main:app --reload
```

### 2. 运行 API 集成测试

```bash
cd backend
python3 test_api.py
```

**预期结果**：所有测试通过，显示 ✅

### 3. 运行性能测试

```bash
cd backend
python3 test_performance.py
```

**预期结果**：平均响应时间 < 500ms

### 4. 启动前端服务

```bash
cd frontend
npm run dev
```

### 5. 手动功能测试

访问 http://localhost:5173，按照测试计划逐项测试：

#### 5.1 登录测试
- 用户名：admin
- 密码：admin123

#### 5.2 学员管理测试
- 查看学员列表
- 展开学员行，查看薄弱项标签
- 点击标签切换颜色（绿→黄→红）
- 添加新薄弱项
- 删除薄弱项

#### 5.3 督学日志测试
- 新增督学日志
- 点击常用短语快速插入
- 选择联系方式、心情、学习状态
- 设置下次跟进日期

#### 5.4 作业管理测试
- 新增作业，填写题量和模块
- 查看作业列表，确认新字段显示
