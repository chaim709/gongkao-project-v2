## 二、功能模块

### 2.1 数据看板（首页）
- **统计卡片**：学员总数、活跃学员、本月督学日志、今日打卡
- **趋势图**：支持切换 7天/14天/30天 查看打卡、新增学员、督学日志趋势
- **分布图**：学员状态分布、报考类型分布、心情分布

### 2.2 学员管理
**查看列表**
- 支持按姓名、手机号搜索
- 支持按状态筛选（在读/暂停/已结业）
- 点击"详情"查看学员完整信息

**新增学员**
1. 点击"新增学员"按钮
2. 填写必填项：姓名
3. 填写选填项：手机号、微信、学历、专业、报考类型等
4. 选择督学老师
5. 提交

**编辑/删除**
- 编辑：点击操作列"编辑"按钮
- 删除：仅管理员可操作，点击"删除"后需二次确认（软删除）

### 2.3 督学日志
**记录日志**
1. 进入"督学日志"页面
2. 点击"新增日志"
3. 选择学员、填写日期、沟通方式
4. 选择心情（积极/稳定/焦虑/低落）和学习状态
5. 填写沟通内容
6. 设置下次跟进日期
7. 提交

**跟进提醒**
- 切换到"跟进提醒"标签页
- 系统自动列出超过 7 天未跟进的学员
- 点击"去跟进"快速创建���学日志

### 2.4 课程管理
- 查看所有课程列表
- 新增/编辑课程（名称、类型、授课老师、起止日期）
- 删除课程（仅管理员）

### 2.5 作业管理
**发布作业**
1. 选择关联课程
2. 填写作业标题和描述
3. 设置截止日期
4. 发布

**批改作业**
1. 查看学员提交的作业
2. 填写分数（0-100）和评语
3. 提交批改

### 2.6 打卡管理
- 查看学员打卡记录
- 查看打卡排行榜
- 查看打卡统计数据

### 2.7 岗位管理
**添加岗位**
- 填写岗位名称、部门、地点、学历要求、专业要求等

**智能匹配**
1. 选择学员
2. 系统根据学员的学历、专业、户籍、报考类型等信息
3. 自动匹配推荐岗位并给出匹配度分数

### 2.8 审计日志（仅管理员）
- 查看所有操作记录
- 支持按操作类型、资源类型筛选
- 记录操作人、时间、内容

## 三、常见操作流程

### 3.1 新学员入学流程
1. 学员管理 → 新增学员 → 填写基本信息
2. 分配督学老师
3. 督学老师首次沟通并记录督学日志
4. 为学员安排课程

### 3.2 日常督学流程
1. 查看"跟进提醒"，了解哪些学员需要跟进
2. 与学员沟通（电话/微信/面谈）
3. 记录督学日志（心情、学习状态、沟通内容）
4. 设置下次跟进时间

### 3.3 作业布置流程
1. 课程管理 → 选择课程
2. 作业管理 → 发布作业
3. 等待学员完成
4. 批改作业并给出评语

---

## 四、注意事项

1. **数据安全**：删除操作为软删除，数据不会真正丢失
2. **权限控制**：删除学员、删除课程等敏感操作仅管理员可执行
3. **浏览器要求**：推荐使用 Chrome、Edge 或 Safari 最新版
4. **手机号格式**：必须为 11 位有效手机号（1开头）
